# My Neovim Configure

## Features

- Packer manager plugins
- Nice UI, nice notify
- Highlight
- Auto comment 
- File Expolre tree
- LSP support
- Dap support
- ...

## Useage 


```
git clone https://github.com/Wananbrstl/nvim.git ~/.config/nvim
```
 
need neovim version `>=0.8`

then `neovim` open neovim, wait for second, the `packer` auto install plugins.
